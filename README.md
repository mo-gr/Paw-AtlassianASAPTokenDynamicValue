# Atlassian ASAP Token Dynamic Value extension for Paw

Based on a fork of https://github.com/luckymarmot/Paw-JsonWebTokenDynamicValue

## Development

### Prerequisites

```shell
npm install
```

### Build

```shell
npm run build
```

### Install

```shell
make install
```

### Test

```shell
npm test
```
